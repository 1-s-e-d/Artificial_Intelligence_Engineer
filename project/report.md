# Отчёт по проекту: Regression Modeling — LightGBM Forecasting

> Полный обзор проекта находится в [`documentation/project_overview.md`](project_overview.md).
> Данный файл — выжимка ключевых разделов для быстрого ознакомления.

---

## 1. Задача

Прогнозирование почасового электропотребления для энергосбытовых и генерирующих компаний.
**Тип:** многомерная регрессия временного ряда с экзогенными переменными.
**Горизонт:** 24 шага вперёд (суточный профиль).

---

## 2. Данные

| Параметр | Значение |
|---|---|
| Объём | 30 816 записей |
| Период | 01.01.2019 — 07.07.2022 |
| Целевая | `Ypowerconsumption` (МВт) |
| Признаки | hour, temp, daylength, cloud, wet, winddir, windspeed, sunrise, sunset, day/night, workday |
| Среднее значение | 181.6 МВт |
| Стандартное откл. | 41.5 МВт |

**Предобработка:** удаление 11 выбросов (1.5×IQR), проверки ADF/KPSS/VIF/гетероскедастичности.

---

## 3. Модель

**LightGBM** (gradient boosting, leaf-wise, histogram algorithm, GOSS).
Доверительные интервалы: **MAPIE** (conformal prediction).
CPU/GPU выбор автоматически через бенчмарк (`project/configs/device_config.json`).

---

## 4. Эксперименты

| Модель | MAPE  | WAPE  | R²    |
|---|-------|-------|-------|
| Ridge Regression | ~8%   | ~8%   | ~0.82 |
| LightGBM (дефолт) | ~1,8% | ~1,7% | ~0.91 |
**Валидация:** скользящее окно (`SlidingWindowBifurcation`), временное разбиение без перемешивания.

---

## 5. Результаты

LightGBM хороша по всем метрикам. Инференс на 24 шага.
Ключевые признаки: `hour`, `daylength`, `temp`.
