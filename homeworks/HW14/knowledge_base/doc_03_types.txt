Основные встроенные типы данных Python: int, float, complex, str, list, tuple, set, frozenset, dict, bool, bytes, bytearray.
Python — язык с динамической типизацией.
Тип переменной определяется автоматически при присваивании значения.
Функция type() возвращает тип объекта, isinstance() проверяет тип.
Начиная с Python 3.5 поддерживаются аннотации типов через модуль typing.
