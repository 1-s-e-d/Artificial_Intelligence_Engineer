Функции в Python определяются ключевым словом def.
Lambda — анонимная однострочная функция: lambda x: x * 2.
Python поддерживает функции высшего порядка, замыкания и декораторы.
Аргументы: позиционные, именованные (keyword), *args для произвольного числа, **kwargs для именованных.
Значения по умолчанию для аргументов задаются в сигнатуре функции.
Функции в Python являются объектами первого класса.
