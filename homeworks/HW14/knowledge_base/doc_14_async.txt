Асинхронное программирование в Python реализовано через модуль asyncio.
Корутина определяется через async def, вызывается через await.
await приостанавливает выполнение корутины до получения результата.
Event loop (цикл событий) управляет планированием и выполнением корутин.
asyncio.gather() запускает несколько корутин параллельно.
Асинхронность эффективна для I/O-bound задач: сетевые запросы, работа с файлами.
