f-строки (f-strings, formatted string literals) введены в Python 3.6.
Синтаксис: f'текст {выражение}' — выражение вычисляется при создании строки.
Поддерживают форматирование: f'{value:.2f}' для float, f'{value:>10}' для выравнивания.
Можно вызывать методы и функции внутри: f'{name.upper()}'.
В Python 3.12 улучшена поддержка вложенных выражений.
f-строки быстрее метода .format() и оператора % форматирования.
