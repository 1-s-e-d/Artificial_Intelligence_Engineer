Модуль json позволяет сериализовать и десериализовать данные в формате JSON.
json.dumps() преобразует объект Python в строку JSON.
json.loads() разбирает строку JSON и возвращает объект Python.
json.dump() и json.load() работают непосредственно с файловыми объектами.
Поддерживаемые типы: dict, list, str, int, float, bool, None.
Параметр indent задаёт форматирование с отступами для читаемого вывода.
