Модуль dataclasses введён в Python 3.7 для удобного создания классов данных.
Декоратор @dataclass автоматически генерирует методы __init__, __repr__, __eq__.
Поля класса определяются через аннотации типов на уровне класса.
Функция field() позволяет задать значения по умолчанию и дополнительные метаданные.
Параметр frozen=True создаёт неизменяемый (immutable) dataclass.
Параметр order=True добавляет методы сравнения __lt__, __le__, __gt__, __ge__.
